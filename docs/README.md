# Project Documentation

This repository hosts a Jekyll site that curates Docker Desktop Extensions. While there are no traditional code libraries or SDKs here, the site itself exposes public configuration and layout components that contributors interact with.

- See configuration options: `docs/configuration.md`
- Learn about the site layout and how to use it: `docs/layouts.md`
- Author and maintain content: `docs/content-authoring.md`
- Run the site locally for preview: `docs/local-development.md`
- FAQs and operational tips (examples included): `docs/faq-usage.md`

If you’re looking to add or update extension entries and tables in the root `README.md`, start with `docs/content-authoring.md`.
